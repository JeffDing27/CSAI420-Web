# 1. Executive Summary

## Overview
The CSAI 420 STEDI Project adds AI-powered features, Voice IVR, and Human-in-the-Loop review capabilities to the existing STEDI fall-risk assessment platform. 

## Key Features Implemented (Weeks 4-7)
- **Mobility Coach Escalation (Week 4):** A triage system for user questions that forwards to human reviewers via AWS SQS.
- **Chat-Assisted Registration (Week 5):** An interactive registration flow reducing drop-off rates using conversational AI.
- **RAG & Memory Integration (Week 6):** LangGraph agents equipped with STEDI-specific knowledge and session memory.
- **Moderator Dashboard (Week 6):** A web UI for human reviewers to manage and resolve escalations.
- **Twilio Voice IVR (Week 7):** Phone-based support for seniors preferring voice interaction, with automatic routing based on intent.
