# 10. Moderator Dashboard Guide

## Accessing the Dashboard
Navigate to `/moderator` in the web application. 

## Capabilities
1. **View Escalations:** See a real-time list of all user questions or registration issues flagged by the AI.
2. **Prioritization:** Escalations are color-coded (Red for High priority/Medical).
3. **Status Management:** Update status from `Escalated` to `Assigned` to `Resolved`.
